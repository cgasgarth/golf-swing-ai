# Dashboard Mockup Prompt

High-fidelity desktop app UI mockup, professional sports-tech aesthetic, dark mode.

**Layout:**
- **Main View:** Large video player featuring a golf swing with neon-colored skeletal pose overlays and club path tracking lines.
- **Timeline:** Horizontal video scrub bar with tagged swing phases (Address, Top, Impact, Follow-through).
- **Metrics Panel:** Sidebar with "TrackMan-style" data grids: Club Speed, Ball Speed, Launch Angle, and Spin Rate in bold monospace typography.
- **AI Insights:** A dedicated "Gemma AI" panel with a chat-like interface providing concise swing tips and recommended drills.

**Style:**
- Deep charcoal and navy background, electric lime and cyan accents for data.
- Minimalist sans-serif typography, high contrast, glassmorphism effects on panels.
- 4K resolution, cinematic lighting, clean professional SaaS interface.
